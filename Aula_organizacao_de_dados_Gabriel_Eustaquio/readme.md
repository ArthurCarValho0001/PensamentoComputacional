# Estrutura 1: Lista

A estrutura de lista representa elementos organizados em sequência linear.

Arquivo:
- Lista.puml

Exemplo:
Lista de alunos matriculados.

---

# Estrutura 2: Grafo

O grafo representa conexões entre departamentos da universidade.

Arquivo:
- Grafo.puml

Exemplo:
Relacionamentos entre Administração, RH, TI e Financeiro.

---

# Estrutura 3: Hierarquia

A hierarquia representa níveis organizacionais da universidade.

Arquivo:
- Hierarquia.puml

Exemplo:
Estrutura organizacional da universidade.

---

# Conclusão

Cada estrutura representa o mesmo conjunto de informações de maneiras diferentes:

- Lista → ordem/sequência
- Grafo → conexões
- Hierarquia → níveis de subordinação

Essas estruturas são fundamentais para organização de dados e desenvolvimento computacional.
